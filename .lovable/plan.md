

# Portfólio de Ciência de Dados 🎓

Um site pessoal com design **dark mode**, visual tech e profissional, focado em destacar seus projetos de ciência de dados.

---

## Estrutura do Site (Single Page)

### 1. Hero Section
- Seu nome, título ("Cientista de Dados" ou similar) e uma frase curta de apresentação
- Links para **GitHub** e **LinkedIn** como ícones
- Animação sutil de entrada

### 2. Sobre Mim (breve)
- Parágrafo curto sobre você, mencionando que está concluindo a faculdade de Ciência de Dados
- Tech stack principal exibida como badges/chips (Python, SQL, Machine Learning, etc.)

### 3. Projetos (seção principal)
- **Galeria em cards** com visual destacado
- Cada card mostra: título, descrição curta, tags de tecnologias usadas, e links (GitHub / demo)
- **Filtro por categoria** (ex: Machine Learning, Análise de Dados, Visualização, etc.)
- Layout em grid responsivo

### 4. Contato
- Email para contato
- Links do GitHub e LinkedIn repetidos
- Visual simples e direto

---

## Design
- **Dark mode** com fundo escuro e detalhes em azul/roxo
- Tipografia limpa e moderna
- Animações sutis de scroll
- Totalmente responsivo (mobile, tablet e desktop)
- Navegação suave com scroll para seções
